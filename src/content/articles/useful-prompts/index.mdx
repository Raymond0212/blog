# Useful Prompts
This section is a collection of prompt and policy reference files for agent-driven development workflows. The files here are intended to be reused as operating templates for orchestration, repository grounding, verification, and documentation discipline.

## Table of Contents
- **Codex Orchestrator Prompt**: A workflow prompt that defines a verification-first orchestration loop using specialized subagents for coverage, completeness, implementation, and documentation.
- **Repository Knowledge Policy**: A structured policy prompt describing how repository knowledge should be organized, maintained, and treated as source-of-truth for humans and agents.
